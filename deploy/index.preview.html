Preview site locally

This workspace contains two site copies: a portfolio version and an html variant for testing.

If you started a local HTTP server in the project root, you'll be redirected to /portfolio/index.html automatically. You can also choose one of these links:

  • Open portfolio/index.html
  • Open html/index.html

Start a local server (recommended)

Run one of these commands in PowerShell:

cd "c:\Users\Admin\OneDrive\Documents\codenguhoc\portfolio"
python -m http.server 8000 --bind 127.0.0.1

# or to preview both simultaneously:
# cd "c:\Users\Admin\OneDrive\Documents\codenguhoc\html"
# python -m http.server 8001 --bind 127.0.0.1

If you prefer running the server from the repo root, open /portfolio/index.html after starting the server.

If links still 404, check you started the server in the folder containing the HTML file (e.g. `portfolio`).
